Benodigdheden:
-Alle bestanden
-Directe connectie met database
-XAMPP

Wat je moet doen om de site te laten werken:
-Ga naar https://www.apachefriends.org/index.html en installeer XAMPP
-Kies alle basisopties in de installer
-Wanneer XAMPP is geïnstalleerd, moet je in de map 'htdocs' alle bestanden verwijderen en alle bestanden van de map 'htdocs' in GitHub erin zetten
-Krijg een directe connectie met de database, bijvoorbeeld via een switch
-Zodra je bent aangesloten op de switch, moet je je IP-adres en Subnetmasker bij de adapter instellingen wijzigen, zodat het aansluit bij de Raspberry Pi met de database erop
-Start XAMPP op en druk op start bij 'Apache'
-Druk nu op 'Admin' en je zit op de website!
